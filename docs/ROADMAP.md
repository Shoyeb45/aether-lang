# Roadmap

## v0.1.0 — Core Language (Released)

- [x] Primitive types: `number`, `boolean`, `string`, `nil`
- [x] Arithmetic operators: `+` `-` `*` `/`
- [x] Comparison operators: `==` `!=` `<` `>` `<=` `>=`
- [x] Logical operators: `AND`, `OR`
- [x] Unary operators: negation (`-`), not (`!`)
- [x] Clean, line-numbered syntax error messages
- [x] Console output (`print`)
- [x] Lexical scoping: global, local, and block scope
- [x] Control flow: `if` / `else` / `else if`
- [x] Looping: `while`, `for`
- [x] Custom functions, higher-order functions, native functions (`clock()`)
- [x] Closures
- [x] Variable resolution and binding pass
- [x] Classes: declaration, instantiation, methods, constructors
- [x] Inheritance, method overriding, `super`, `this`
- [x] Cross-platform release binaries (Linux, macOS, Windows)

## Near-Term

- [ ] Standard library basics: string methods, math functions, array/list type
- [ ] Better runtime error messages (stack traces, not just line numbers)
- [ ] `break` / `continue` in loops
- [ ] Automated test suite (interpreter-level and language-level)
- [ ] Package the interpreter for common package managers (Homebrew, apt, winget)

## Longer-Term / Exploratory

- [ ] Async/concurrency primitives
